# 项目介绍

这个项目是一个自用的模板项目，方便直接拿来做后续页面的开发。项目采用了以下技术：

- Vue3
- Vue Router
# simple-vue

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
